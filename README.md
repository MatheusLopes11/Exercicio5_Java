Neste exercício, utilizei o JShell para trabalhar com arrays, estruturas de repetição e conversão de unidades de medida,
simulando o processamento de um conjunto de medições científicas e a geração de um relatório formatado.

Conceitos aplicados:

- Declaração e inicialização de arrays
- Arrays de tipos primitivos (double[])
- Arrays de tipos de referência (String[])
- Utilização da propriedade length
- Acesso e modificação de elementos por índice
- Estruturas de repetição:
- while
- for
- Controle de iteração através de índices
- Declaração de constantes com final
- Operadores aritméticos
- Operadores de atribuição
- Concatenação de Strings
- Caracteres de escape (\n e \t)
- Construção de relatórios textuais dinâmicos
- Reatribuição de arrays
- Conversão de unidades de medida
- Expressão switch moderna com sintaxe ->
- Manipulação simultânea de múltiplos arrays relacionados
- Processamento e transformação de dados armazenados em estruturas de dados
- O que foi desenvolvido

Durante a prática, foram criados arrays para armazenar:

- Descrições das medições
- Valores numéricos
- Unidades de medida

Inicialmente, os dados foram organizados em um relatório textual utilizando estruturas de repetição. Em seguida, foi implementado
um processo de conversão automática das unidades do sistema imperial para o Sistema Internacional (SI), incluindo:

- Fahrenheit → Kelvin
- Pé (ft) → Metro (m)
- Libra (lb) → Quilograma (kg)

Para realizar essas conversões, utilizei a expressão switch, aplicando diferentes fórmulas e fatores de conversão de acordo com a
unidade associada a cada medição.

Ao final, os dados convertidos foram utilizados para gerar um novo relatório formatado contendo as medições já normalizadas para o
Sistema Internacional.

Aprendizados:

Esta prática reforçou conceitos fundamentais de Java relacionados a arrays, iteração com while e for, manipulação de Strings,
uso da propriedade length, controle de índices e transformação de dados. Também permitiu compreender melhor como percorrer
estruturas de dados para processar e atualizar informações de forma automatizada, uma habilidade essencial para o desenvolvimento
de aplicações mais complexas.
